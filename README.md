Sistema de Gestión de Turnos de Estudiantes

Descripcion
Este programa en C++ permite gestionar turnos de atencion de estudiantes en consola  
Se pueden registrar, atender, mostrar y buscar estudiantes

Estructura usada y justificación
Se implementó lista enlazada simple manualmente (nodos y punteros)  
Justificación: permite inserción eficiente al final y eliminación al inicio (FIFO), cumpliendo la lógica de turnos

Complejidad
- Inserción: O(1) al final  
- Eliminación: O(1) al inicio  
- Bsqueda: O(n) por recorrido secuencial  

Instrucciones para la compilación y ejecucion
El código fue probado en "Visual Studio Code" con extensión de compilador ya que la computadora de la Universidad no cuenta con el compilador
